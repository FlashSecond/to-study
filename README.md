# to-study v5.0 — 知识深度教材生成技能

> 把任何内容（网页、文档、书籍、API 参考）转化为**专业教学级深度教材**。
> v5.0 革新：8 模块结构 + 三列设计哲学 + 学习心理学驱动。

---

## 🎯 一句话

输入一份资料 → 源书读取 → 8模块编撰 → 十二维审核 → 输出深度教材：每章含"一目了然"（可扫读）和"快速查找"（可检索）两栏，即学即查。

## 📐 8 模块结构

```
模块1  5分钟极速上手卡    零门槛 Hello World → 5分钟正反馈
模块2  结构全景图          所见即所得的功能/结构导航
模块3  核心概念与架构      术语表+模型图+机制+设计理念
模块4  任务导向操作指南    卡片式任务，按频次排序，基础→组合→综合
模块5  功能完整参考手册    三重索引的字典式查阅
模块6  实战案例库          按行业/岗位组织的端到端工作流
模块7  排障诊所            陷阱+FAQ+故障排查树+错误码词典
模块8  练习与自测          分级练习+动手实验+标签化检索
───   全局速查工具箱      独立中枢层：快捷键+标签云+同义词映射
```

```

     ↓              ↓                  ↓

                                                      ↓
                                                  ✅ 教材交付
```

## ⚙️ 核心红线（9 条强制标准）

| # | 红线 | 阈值 |
|:--:|------|:--:|
| 1 | 8 模块完整 | 缺失 → 打回 |
| 2 | 字数达标 | 核心 ≥8000 字 / 普通 ≥5000 字 |
| 3 | 代码示例 | ≥5 个 |
| 4 | Mermaid 图 | ≥3 张 |
| 5 | 功能/API 矩阵 | framework/tool 必含 |
| 6 | 结构-API 映射图 | framework/tool 必含 |
| 7 | 陷阱+排障 | ≥5 陷阱 + FAQ + 排障树 |
| 8 | 练习含答案 | ≥6 题 + 动手实验 |
| 9 | 附录完整 | 快捷键+术语对照+资源索引 |

## 🏷️ 6 种内容类型

| 类型 | 标签 | 拆分策略 |
|:---:|------|------|
| 🖥️ 编程框架/库 | `framework` | 按模块/命名空间 |
| 🔧 工具/软件 | `tool` | 按功能模块 |
| 🧠 理论/方法论 | `theory` | 按概念层级 |
| 📖 教程/课程 | `tutorial` | 按课时/难度递进 |
| 📚 书籍/长文 | `book` | 按原有章节 |
| 🗃️ 综合/多主题 | `mixed` | 按主题聚类 |

## 🔍 十二维审核

| 维度 | 权重 | 维度 | 权重 |
|------|:---:|------|:---:|
| 内容深度 | 16% | 代码质量 | 16% |
| 结构完整(8模块) | 13% | 图表丰富 | 10% |
| API 矩阵质量 | 8% | 结构映射质量 | 8% |
| API 覆盖 | 8% | 陷阱+排障质量 | 8% |
| 练习质量 | 6% | 教学例子 | 4% |
| 中文表达 | 2% | 技术准确性 | 1% |

**等级**：A≥90 ✅ 放行 | B≥80 ✅ 放行+建议 | C≥70 ⚠️ 有条件 | D<70 ❌ 打回
