## 📐 8 模块结构

```
模块1  5分钟极速上手卡       零门槛 Hello World → 正反馈
模块2  结构全景图             定义+价值+全景图+同类对比
模块3  核心概念与架构         术语+模型图+机制+设计理念+结构映射
模块4  任务导向操作指南       卡片式任务，基础→组合→综合
模块5  功能参考手册           三重索引的字典式查阅
模块6  实战案例库             按行业/岗位组织的端到端工作流
模块7  排障诊所               陷阱+FAQ+决策树+错误码词典
模块8  练习与自测             分级练习+动手实验，答案折叠
───   全局速查中枢            独立中枢层：快捷键+标签云+同义词映射
```

## 🔄 七阶段流水线

```
Stage 1 采集 → Stage 2 结构规划 → Stage 3 知识提炼
     ↓              ↓                  ↓
Stage 4 编撰（子代理一章一Agent并行）→ Stage 5 排版 → Stage 6 中枢构建 → Stage 7 审核
                                                                               ↓
                                                                           ✅ 交付
```

**阶段 4 编撰模式**：一章一子 Agent（`sessions_spawn`），每批 2-4 Agent 并行，主 Agent 验证后继续下批。子 Agent 直接 `write` 输出 `.md` 终稿。

---

## ⚙️ 核心红线（9 条强制标准）

| # | 红线 | 阈值 |
|:--:|------|:--:|
| 1 | 8 模块完整 | 缺失任何模块 → 打回 |
| 2 | 字数达标 | framework≥30000字/章；tool/theory≥20000字/章 |
| 3 | 代码/示例充分 | framework/tool≥5可运行示例；theory≥5论证示例 |
| 4 | Mermaid 图 | ≥3 张/章 |
| 5 | 功能矩阵 | 功能分类/签名/参数/★常用度/使用前提/关联功能 |
| 6 | 结构-功能映射 | 文本标注图 + Mermaid 映射图 |
| 7 | 陷阱+排障决策树 | ≥5 陷阱（❌/✅+现象+根因）+ Mermaid 决策树 |
| 8 | 练习含答案 | ≥6 题（基础≥3+进阶≥2+挑战≥1），答案折叠 |
| 9 | 全局速查中枢 | 快捷键+术语对照+标签云+同义词映射+搜索索引 |
